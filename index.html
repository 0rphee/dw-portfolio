Creé este programa para generar mis horarios de cada semestre solamente especificando las clases que deseo tomar y los profesores disponibles de cada materia.

ia-vuelos

Proyecto final de la clase de Inteligencia Artificial: un servicio búsqueda de vuelos de avión entre aeropuertos para una fecha determinada. Se usa información real de aeropuertos y aviones, pero los vuelos en sí son autogenerados previamente.

La generación de vuelos es realizada por varios scripts de Python que llenan una una base de datos de MySQL. La búsqueda en sí se hace en un servidor de Flask, con un frontend escrito en Elm. Se usa Docker para el manejo de los servicios localmente.
