También me gusta mucho trabajar en proyectos de código propios para resolver problemas que tenga o simplemente investigar algún tema que me interese. Me agradan mucho los lenguajes funcionales, y siempre que puedo hago mis proyectos en alguno de estos, como Haskell, Elm o Rust. Sin embargo, a futuro también me gustaría aprender y trabajar con C.

Proyectos

schedule-maker

Programa de línea de comando para generar horarios de clases sin conflictos a partir de un archivo .yaml. Genera calendarios (archivos .ics para Google Calendar y iCalendar de Apple) y archivos de Excel (.xlsx), con todos los horarios válidos.

Creé este programa para generar mis horarios de cada semestre solamente especificando las clases que deseo tomar y los profesores disponibles de cada materia.

Haskell Repositorio
prop-logic-paevty

Programa de línea de comando para crear tablas de verdad para typst a partir de fórmulas proposicionales.

La funcionalidad original fue escrita en Python pero cuando necesité más funcionalidad lo reescribí en Haskell. Lo usé para verificar y generar tablas de verdad de las tareas de mi clase de Lógica Proposicional.

Haskell Typst Repositorio
ia-vuelos

Proyecto final de la clase de Inteligencia Artificial: un servicio búsqueda de vuelos de avión entre aeropuertos para una fecha determinada. Se usa información real de aeropuertos y aviones, pero los vuelos en sí son autogenerados previamente.

La generación de vuelos es realizada por varios scripts de Python que llenan una una base de datos de MySQL. La búsqueda en sí se hace en un servidor de Flask, con un frontend escrito en Elm. Se usa Docker para el manejo de los servicios localmente.
